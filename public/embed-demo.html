🚀 Formulario Embebido Alora

Demo de integración en sitios de terceros

📄 Formulario Inline

Se inserta directamente en cualquier página web.

<script src="http://localhost:3000/embed.js" data-form-id="default"></script>

(El formulario aparecerá aquí)

💬 Widget Flotante

Botón flotante en la esquina inferior derecha, estilo Intercom.

<script src="http://localhost:3000/embed.js" data-form-id="default" data-mode="widget" data-color="#2d6167"></script>

El widget flotante aparece en la esquina inferior derecha de toda la página

⚡ API Programática

Control total desde JavaScript para integraciones personalizadas.

// Abrir / cerrar / toggle window.AloraLeadForm.open() window.AloraLeadForm.close() window.AloraLeadForm.toggle() // Disparar desde cualquier botón <button onclick="window.AloraLeadForm.open()">Contactar</button>
